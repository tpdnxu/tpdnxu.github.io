VSCode配置PHP除錯工具XDebug

發表於 2024-09-23 分類於 應用工具

操作步驟

  1. 開啟 Installation Wizard 頁面
  2. 打開命令提示字元執行 php -i | clip 複製PHP信息後貼上
  3. 下載Xdebug擴充模組 php_xdebug-3.3.1-8.1-vs16-nts-x86_64.dll
  4. 移動下載完成的擴充模組至PHP安裝路徑下的ext目錄
  5. 編輯PHP配置檔並加入Xdebug擴充模組
  6. 編輯VSCode的PHP配置(開啟VSCode設定/Ctrl+,搜尋PHP後設置指向PHP可執行檔)
  7. 使用VSCode開啟專案目錄點擊執行選擇新增組態或在除錯模式建立 launch.json
